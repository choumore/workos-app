# Career Context

Read `workos-state/vectors.md`, `workos-state/wins.md`, and `workos-state/coaching.md` from the vault.

## What to do

1. **Summarize current state** — For each vector, report tier (Strong/Building/Thin/No evidence), score, and days since last evidence
2. **Identify gaps** — Flag vectors that are Thin or have no evidence in 14+ days
3. **Cross-reference coaching** — Check if any active coaching observations (especially escalation level 4+) relate to gap vectors
4. **Assess promotion readiness** — Based on vector distribution: how many are Strong vs Building vs Thin
5. **Recommend next steps** — 2-3 specific, actionable items that would strengthen the weakest vectors

## Output format

```
## Career Snapshot

**Overall:** X of Y vectors at Strong, Z Building, W need attention

### Vector Status
- [Vector Name]: [Tier] — [score] pts, last evidence [N days ago]
  ...

### Gaps & Opportunities
- [Specific gap with actionable suggestion]

### Coaching Patterns
- [Active observation] (escalation: N/7, surfaced X times)

### Recommended Actions
1. [Specific action tied to a vector]
2. ...
```

## Tone

Direct and constructive. This is a career dashboard readout, not a pep talk. Be honest about gaps while framing them as opportunities.
