# Visibility Draft

Read `workos-state/wins.md`, `workos-state/vectors.md`, and `workos-state/priorities.md` from the vault.

## What to do

Generate a first-person visibility post (WAYWO or Top 3 format) using real evidence from WorkOS.

### WAYWO format (What Are You Working On)
Use when asked for a general update. Pull from recent wins and current priorities.

### Top 3 format
Use when asked for a weekly Top 3. Pull from the weekly plan's Top 3 items and recent progress.

## Output format

### WAYWO

```
Here's what I've been focused on:

**[Vector/Theme 1]**
[2-3 sentences about specific work, referencing actual wins/entries]

**[Vector/Theme 2]**
[2-3 sentences]

**Coming up:** [1-2 forward-looking items from priorities]
```

### Top 3

```
Top 3 this week:

1. **[Item from weekly plan]** — [progress/outcome, reference specific wins]
2. **[Item]** — [progress]
3. **[Item]** — [progress]
```

## Rules

- Always write in first person (I/my)
- Use specific details from wins and entries — never generic filler
- Keep it conversational but professional
- Reference impact where wins have it
- If there aren't enough wins for a full post, say so and suggest waiting or focusing on specific areas
- Present as a draft for Mo to edit, not a final post
