# Debrief Enrichment

Read `workos-state/vectors.md` and `workos-state/coaching.md` from the vault.

## What to do

After the regular daily debrief, append a **Career Reflection** section:

1. **Evidence collected today** — Which vectors got new evidence from today's work? (map debrief items to vectors)
2. **Missed opportunities** — Were there meetings or tasks where a vector could have been strengthened but wasn't?
3. **Coaching check-in** — Did today's work address any active coaching observations?
4. **Win candidates** — Did anything from today qualify as a win worth logging in WorkOS?

## Output format

Append this to the debrief:

```
---

## Career Reflection

**Evidence today:** [Vector A] got strengthened by [specific work]. [Vector B] got a signal from [meeting/task].

**Missed opportunity:** [Meeting X] touched on [Vector C] territory — could have [specific action].

**Win candidate:** [Description of something that could be logged as a win]
```

## Rules

- Be specific — reference actual work items from the debrief
- Only flag missed opportunities when they're real and actionable, not hypothetical
- Max 4-5 lines
- If it was a low-key day with no vector relevance, say so in one line and move on
